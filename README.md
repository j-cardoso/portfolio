# Portfolio Next.js Moderno 🚀

Portfolio profissional desenvolvido com as tecnologias mais modernas do ecossistema React.

## ✨ Funcionalidades

- ✅ **Design moderno e responsivo** - Funciona perfeitamente em todos os dispositivos
- ✅ **Tema escuro/claro** - Alternância suave entre temas com animações
- ✅ **Internacionalização (i18n)** - Suporte completo para Português e Inglês
- ✅ **Animações profissionais** - Transições suaves com Framer Motion
- ✅ **SEO otimizado** - Metadados configurados para melhor indexação
- ✅ **Performance otimizada** - Next.js 15 com App Router
- ✅ **TypeScript** - Tipagem estática para melhor experiência de desenvolvimento

## 🚀 Tecnologias

- **Next.js 15** - Framework React com App Router e React 19
- **TypeScript** - Tipagem estática
- **Tailwind CSS** - Framework CSS utilitário moderno
- **Framer Motion** - Biblioteca de animações profissionais
- **Lucide React** - Ícones modernos e leves

## 🛠️ Como executar

### 1. Instalar dependências:

```bash
npm install
```

### 2. Executar em modo desenvolvimento:

```bash
npm run dev
```

### 3. Abrir no navegador:

Acesse [http://localhost:3000](http://localhost:3000)

## 📦 Build para produção

```bash
npm run build
npm start
```

## 🎯 Deploy na Vercel (Recomendado)

1. Faça push do código para o GitHub
2. Importe o projeto na [Vercel](https://vercel.com)
3. Deploy automático!

## 📝 Personalização

### Alterar traduções:
Edite `components/language-provider.tsx` e atualize o objeto `translations`.

### Modificar cores e tema:
- **Cores principais**: `app/globals.css` (variáveis CSS)
- **Tema Tailwind**: `tailwind.config.ts`

### Atualizar conteúdo:
Edite os componentes em `components/sections/`
